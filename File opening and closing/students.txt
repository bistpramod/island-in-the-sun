Python is a high-level programming language that is designed to be easy to read and understand. Because of this, it is often used by beginners. Two very important concepts in Python are integers and functions. These concepts form the foundation of programming and are used in almost every Python program.

Integers in Python

Integers are whole numbers. They do not have decimal points. In Python, integers can be positive, negative, or zero. Examples include numbers like 5, −10, and 0.

Python automatically recognizes integers when they are assigned to a variable. This means you do not need to tell Python that a number is an integer. Python figures it out on its own.

Integers are commonly used to represent things like counts, scores, ages, and quantities. Since many real-world problems involve counting or measuring whole numbers, integers are one of the most important data types in Python.

Variables and Integers

A variable is used to store data in a program. When you assign an integer to a variable, Python stores that value in memory and allows you to use it later.

Python uses dynamic typing, which means you do not need to declare the type of a variable. The type is decided automatically based on the value stored in it.

Integer Operations

Python allows you to perform many mathematical operations using integers.

You can add, subtract, multiply, and divide numbers. Division usually produces a decimal result, even if both numbers are integers. Python also provides floor division, which removes the decimal part and gives only the whole number.

The modulus operation is used to find the remainder after division. This is especially useful for checking whether a number is even or odd. Exponentiation is used to raise a number to a power.

These operations allow Python to handle calculations and solve mathematical problems efficiently.

Type Conversion

Sometimes, you may need to change one data type into another. This is called type conversion or casting.

For example, user input is often stored as text, even if it looks like a number. To perform calculations, the text must be converted into an integer. Likewise, numbers sometimes need to be converted into text so they can be displayed with messages.

Python provides built-in functions to handle these conversions easily.

Functions in Python

A function is a block of code that performs a specific task. Instead of writing the same code multiple times, you can place it inside a function and reuse it whenever needed.

Functions help keep programs organized, readable, and easier to debug. A function only runs when it is called.

Function Parameters and Arguments

Functions can receive input values called parameters. When a function is called, the actual values passed to it are known as arguments.

Using parameters allows a function to work with different values and perform tasks more flexibly.

Return Values

Functions can send data back using a return statement. The returned value can be stored in a variable, printed, or used in further calculations.

Some functions may return more than one value at the same time, which makes them very powerful and useful.

Default Parameters

Functions can also have default parameter values. If no value is provided when calling the function, the default value is used automatically.

This makes functions easier to use and helps prevent errors.

Using Integers Inside Functions

Integers are often used inside functions to perform calculations or make decisions. For example, a function may check whether a number is even or odd or perform mathematical operations and return a result.

Combining integers and functions allows programs to solve real-world problems efficiently.



Python Notes: Integers and Functions
1. Integers in Python

An integer is a whole number. Integers can be:

Positive (e.g., 5, 10)

Negative (e.g., -3, -20)

Zero (0)

In Python, integers belong to the data type called int.
Python can automatically recognize integers when you assign them to a variable.

2. Integer Operations

Python allows many mathematical operations with integers:

Addition: combines two numbers.

Subtraction: finds the difference between numbers.

Multiplication: repeated addition.

Division: divides numbers and returns a decimal value.

Floor Division: divides numbers and removes the decimal part.

Modulus: returns the remainder after division.

Exponentiation: raises a number to a power.

These operations are commonly used in calculations, logic, and problem solving.

3. Type Conversion (Casting)

Type conversion means changing one data type into another.

Common conversions:

String to integer: useful when working with user input.

Integer to string: useful when displaying numbers with text.

Python provides built-in functions like int() and str() for this purpose.

4. Functions in Python

A function is a reusable block of code designed to perform a specific task.

Why functions are important:

Reduce repetition

Make code easier to read

Improve organization

Make debugging easier

A function runs only when it is called.

5. Functions with Parameters

Functions can receive parameters, which are values passed into the function.

Parameters allow functions to:

Work with different data

Be more flexible and reusable

When a function is called, the provided values are called arguments.

6. Return Values

A function can send data back using a return statement.

Returned values can be:

Stored in variables

Used in calculations

Printed or passed to other functions

Some functions may return multiple values at once.

7. Default Parameters

Functions can have default parameter values.

If no argument is provided when calling the function, the default value is used.
This makes functions easier to use and prevents errors.

8. Integers Inside Functions

Integers are often used inside functions for:

Calculations

Conditions

Checking even or odd numbers

Logical decisions

Functions can process integers and return results such as True or False.
